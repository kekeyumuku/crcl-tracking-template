# CRCL Tracking Template

A qualitative framework for tracking whether Circle / CRCL is still mainly a spread-driven stablecoin issuer, or is evolving into a payments infrastructure platform.

## What this is

This repository contains a practical research template for tracking Circle / CRCL.

It is **not** designed to predict short-term price moves.
It is designed to answer a more important question:

> Is CRCL still primarily a spread-driven issuer, or is it gradually becoming a payments infrastructure platform?

## Why this exists

Most company trackers focus on:

- earnings headlines
- market sentiment
- narrative changes
- one-off news

But for Circle, the more important question is not just whether revenue grew.
The real question is **what kind of business is being validated**:

- a rate-sensitive issuer model
- or a higher-multiple infrastructure / platform model

This template helps separate:

- **base economics**
- **upside validation**
- **competitive position**

## What it tracks

The framework tracks 8 core dimensions.

### Base / foundation
1. USDC circulation
2. Growth quality
3. Rate sensitivity
4. Revenue retention / channel economics

### Upside / platform validation
5. Non-spread revenue signals
6. Market share changes
7. Regulatory positioning
8. Enterprise / payments adoption

## Core principle

> **If it doesn’t change the model, don’t promote it to the card.**

Only formalize information if it changes one of these:

- base economics
- upside validation
- competitive position

Do not over-log:

- pure narrative
- roadmap talk
- market mood
- unsupported vision statements

## Who this is for

This template is useful for:

- public market investors tracking CRCL
- crypto / stablecoin researchers
- fintech and payments infrastructure analysts
- anyone studying business-model migration from issuer economics to platform economics

## How to use

Suggested workflow:

1. Read a new piece of information
2. Fill the quick version first
3. Decide whether it deserves a full card update
4. Re-assess whether CRCL now looks more like:
   - a spread-driven issuer
   - a transition story
   - a payments infrastructure platform
